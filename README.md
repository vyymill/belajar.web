# Bahaya Merokok

Situs edukasi singkat tentang bahaya merokok, manfaat berhenti, dan kuiz interaktif.

## Cara menjalankan
1. Buka folder proyek.
2. Buka `index.html` di browser (klik dua kali atau `Open With Live Server`).

## Publikasikan ke GitHub
1. Buka GitHub Desktop.
2. Pilih *File → Add Local Repository* dan arahkan ke folder proyek ini.
3. Klik *Publish repository*, pilih visibility `Public` dan publish.
4. Di GitHub, buka *Settings → Pages* lalu pilih publishing source `main` branch (root). Situs akan tersedia dalam beberapa menit.
